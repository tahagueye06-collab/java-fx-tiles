# JavaFX Tiles

## Beschrijving
Gemaakt als schooloefening in mijn 1e bachelor 
Toegepaste Informatica.

Een interactief speelbord van 4 bij 3 tegels.
Je kan tegels selecteren door erop te klikken
en de tegel onder je muiscursor wordt visueel
gemarkeerd.

## Functies
- Hover effect op tegels met de muis
- Tegel selecteren en deselecteren via muisklik
- Visuele feedback op het canvas

## Technologie
- Java
- JavaFX
- MVP patroon (Model-View-Presenter)
- MouseEvent handlers

## Hoe runnen?
1. Open het project in IntelliJ IDEA
2. Zorg dat JavaFX geïnstalleerd is
3. Run Main.java
4. Beweeg je muis over de tegels en klik om 
   een tegel te selecteren
